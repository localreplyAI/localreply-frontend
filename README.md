# LocalReply Frontend

Frontend statique déployé sur Vercel.

## Structure
- Pages publiques : `home.html`, `pricing.html`, `features.html`, `about.html`, `blog.html`, `contact.html`
- Auth : `auth.html`, `forgot-password.html`, `reset-password.html`
- Dashboard : `dashboard.html`, `dashboard-configuration.html`, `dashboard-rendez-vous.html`, `dashboard-messages.html`, `dashboard-analytics.html`
- Compte : `mon-compte.html`, `abonnement.html`, `onboarding.html`
- Chat client : `chat-client.html`
- Légal : `terms.html`, `privacy.html`, `gdpr.html`

## Deploy
Connecté à Vercel via GitHub. Chaque push sur `main` → deploy automatique.

## Backend
API : `https://localreply-api.onrender.com`
